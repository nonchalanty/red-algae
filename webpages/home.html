KaGE: Kappaphycus alvarezii Genome Explorer

Welcome to KaGE - the Kappaphycus alvarezii Genome Explorer.
Use this platform to browse the complete, assembled genome of Kappaphycus alvarazii, accompanying annotations and perform basic analyses on this data.

For more information about the genome assembly, and to cite this resource, consult the following:
High-quality de novo genome assembly of Kappaphycus alvarezii based on both PacBio and HiSeq sequencing (Jia et al., 2020)

Features

At present, the portal includes the following features:

Interactive Genome Browser

Navigate the complete genome sequence, down to the base pair level. Copy and compare sequences of interest. Optionally select from an array of annotation and alignment tracks.

Rich Feature Annotations

Explore in silico predicted genes and transcripts, alongside their functional annotations. This includes functional gene ontologies as well as pathway annotations by Reactome, MetaCyc and others. Search and navigate between these using the search bar and typing for a pathway ID, GO term or other annotation of interest.

On-the-fly Blast Functionality

Perform Blastn/Blastx searches to aid in evofunctional discovery. Blast searches can be performed directly in the browser by highlighting a sequence of interest and then viewing the results as annotation tracks. Result tracks can be toggled on or off, and "hits" filtered by E-score and other parameters. Blast searches can be performed against both the small, example faux database, as well as the complete JGI Phycocosm (algal) blast database and others. Arbitrary sequences can also be pasted and used in blast searches.

Note: currently, users must be logged in to access blast functionality. Guest users can explore this functionality using the username "user" and password "password".
